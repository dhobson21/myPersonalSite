Burn Book

Burn Book is my front-end capstone project for Nashville Software School. This app demonstrates the front-end skills i've learned while at NSS (HTML, CSS, JavaScript, ReactJS) and demonstrates basic CRUD functionality. Burn-Book is a basic social media app that allowers users to track all of the petty grudges they hold against enemies, frenemies and random strangers. The idea for this app popped into my head and the more I thought about it, the more it made me laugh. I had a blast putting this together and hope users can see the tongue-in-cheek irreverance with which I created this.

Nutshell JS

Nutshell was a group project I contributed to while at Nashville Software School. This app is a dashboard app that allows users to organize their daily tasks, events, news, articles, friends and chat messages. Users can create, edit and delete cards to their dashboard for each category and explore cards beloning to other users once they are friends.

I was personally responsible for all code relating to the tasks component of this project, as well as contributions to the overall layout, friends relationship, chat messages and styling. I really enjoyed working in a group setting on this project, and feel like it improved both my coding abilities and communication.
